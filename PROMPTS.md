# Vibe Coding Prompts

Copy a prompt below, fill in the `[brackets]`, and paste it into Claude.

---

## 1. Start from scratch

> I want to build [describe your app in plain English — e.g., "a landing page for my coffee shop" or "a personal portfolio"]. Set up the project, create the first page with a clean layout, and start the dev server so I can see it.

---

## 2. Add a new page

> Add a new page at /[page-name — e.g., about, contact, pricing] that contains [describe what should be on the page]. Link to it from the main navigation.

---

## 3. Add a section or component

> Add a [hero section / pricing table / FAQ accordion / contact form / testimonials / feature grid] to my [homepage / about page / etc.]. Here is the content: [paste your content or describe it].

---

## 4. Change the visual style

> Redesign the look of my app. Use [describe colors, mood, or style — e.g., "dark mode with purple accents", "clean and minimal", "warm and friendly"]. Update the fonts and spacing too.

---

## 5. Add a feature

> Add [describe the feature — e.g., "a newsletter signup form", "a dark mode toggle", "a product gallery"] to my app. Make it fully functional end to end. Tell me when it's ready and how to use it.

---

## 6. Fix something broken

> Something is not working: [describe what is broken or looks wrong — e.g., "the button doesn't do anything", "the layout looks broken on mobile"]. Find the cause and fix it.

---

## 7. Deploy to Vercel

> Deploy my app to Vercel and give me the live public URL. Make sure everything is working before you deploy.

---

## 8. Review and improve

> Review my app as if you were a user seeing it for the first time. Find 3 things to improve — design, usability, or code quality. Tell me your suggestions, then implement whichever one I choose.
